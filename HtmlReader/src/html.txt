<h1 class="h1">Java URLConnection Class</h1>
<p>The <strong>Java URLConnection</strong> class represents a communication link between the URL and the application. It can be used to read and write data to the specified resource referred by the URL.</p>
<h3 class="h3">What is the URL?</h3>
<ul class="points">
<li>URL is an abbreviation for Uniform Resource Locator. An URL is a form of string that helps to find a resource on the World Wide Web (WWW).</li>
<li>URL has two components:</li>
</ul>
<ol class="points">
<li>The protocol required to access the resource.</li>
<li>The location of the resource.</li>
</ol>
<h3 class="h3">Features of URLConnection class</h3>
<ol class="points">
<li>URLConnection is an abstract class. The two subclasses HttpURLConnection and JarURLConnection makes the connetion between the client Java program and URL resource on the internet.</li>
<li>With the help of URLConnection class, a user can read and write to and from any resource referenced by an URL object.</li>
<li>Once a connection is established and the Java program has an URLConnection object, we can use it to read or write or get further information like content length, etc.</li>
</ol>

<h2 class="h2">Displaying Source Code of a Webpage by URLConnecton Class</h2>
<p>The URLConnection class provides many methods. We can display all the data of a webpage by using the getInputStream() method. It returns all the data of the specified URL in the stream that can be read and displayed.</p>
<h3 class="h3">Example of Java URLConnection Class</h3>
<div class="codeblock"><textarea name="code" class="java">
import java.io.*;  
import java.net.*;  
public class URLConnectionExample {  
public static void main(String[] args){  
try{  
URL url=new URL("http://www.javatpoint.com/java-tutorial");  
URLConnection urlcon=url.openConnection();  
InputStream stream=urlcon.getInputStream();  
int i;  
while((i=stream.read())!=-1){  
System.out.print((char)i);  
}  
}catch(Exception e){System.out.println(e);}  
}  
}
